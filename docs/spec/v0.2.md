# Geno Language Specification v0.2 (Archived)

The historical v0.2 specification is preserved with the
[`v0.3.1` release](https://github.com/davidiach/geno-lang/blob/v0.3.1/docs/spec/v0.2.md).

The maintained normative contract is the
[Geno v0.4 language specification](v0.4.md). The old `v0.2.md` path remains as
an explicit archive pointer so existing documentation links fail informatively
instead of silently presenting a moving specification under the wrong version.
